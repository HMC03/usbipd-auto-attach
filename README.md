# USBIPD Auto Attach

This provides instruction on how to write a windows service that will automatically attatch usb devices to your wsl. Good for serial communication and testing.


1) Install script from this repo

2) Install nssm 

    Download from: https://nssm.cc/download

3) Open Powershell as administrator

4) Run Nssm Gui
```ps
C:path\to\nssm install usbipd-auto-attach
```
